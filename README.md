# Django ORM Homework

Домашнее задание по теме **«Запросы в Django ORM»**.

## Что было сделано

- добавлена новая модель `Genre`;
- в модель `Book` добавлена связь `ManyToManyField` с жанрами;
- модель `Genre` зарегистрирована в административной панели Django;
- через админку добавлены жанры:
  - Фантастика
  - Детектив
  - Роман
- выполнены запросы в Django Shell:
  - поиск всех книг жанра **«Фантастика»**;
  - поиск книг, относящихся одновременно к жанрам **«Фантастика»** и **«Детектив»**;
  - подсчёт количества книг в каждом жанре.

## Результаты запросов

- **Фантастика** — 2 книги
- **Детектив** — 1 книга
- **Роман** — 1 книга
