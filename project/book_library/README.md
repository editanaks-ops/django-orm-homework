# Django ORM Homework — PRO

PRO-часть домашнего задания по теме **«Запросы в Django ORM»**.

## Что было сделано

В проект были добавлены новые модели:

- `Publisher` — издательство;
- `Store` — книжный магазин;
- `Review` — отзывы на книги.

Также была расширена модель `Book`:
- добавлена связь с `Publisher`;
- добавлена связь с `Store`.

## Реализованные связи

- `Publisher` → `Book` — связь **один ко многим** (`ForeignKey`);
- `Book` ↔ `Store` — связь **многие ко многим** (`ManyToManyField`);
- `Book` → `Review` — связь **один ко многим** (`ForeignKey`).

## Выполненные запросы

Были реализованы и проверены запросы:

1. Поиск книг, опубликованных издательствами из России
2. Поиск книг, продающихся в магазинах Москвы
3. Поиск книг со средней оценкой выше 4.5
4. Подсчёт количества книг в каждом магазине
5. Поиск магазинов, где продаются книги, опубликованные после 2010 года

## Оптимизация запросов

- `select_related()` использован для связи `Book -> Publisher`
- `prefetch_related()` использован для связей `stores` и `reviews`
